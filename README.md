## How to Run 🚀

1. Clone the repository:
   ```bash
   git clone https://github.com/hiteshmahto/DSA-Playground.git
   ```
2. Navigate to the problem folder:
   ```bash
   cd repository/Leetcode_Easy
   ```
3. Compile the Java file:
   ```bash
   javac ProblemName.java
   ```
4. Run the program:
   ```bash
   java ProblemName
   ```
